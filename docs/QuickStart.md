# 快速上手

<div class="uku-doc-card">

### 介绍

通过本章节你可以了解到 Uku 的安装方法和基本使用。

## 安装

<div class="uku-doc-card">

通过 npm 安装

```
npm i Uku
```

也可以通过 yarn 安装

```
yarn add Uku
```

</div>

## 引入组件

<div class="uku-doc-card">


```javascript
import { createApp } from 'vue';
import Uku from 'Uku';

const app = createApp();
app.use(Uku);

```

</div>
