### 介绍

Uku 一个在不断完善的vue组件库。

项目地址 [Uku](https://github.com/cq-zz/uku-vue3)

### 安装

```bash
# 通过 npm 安装
npm i Uku

# 通过 yarn 安装
yarn add Uku
```

### 引入组件

```javascript
import { createApp } from 'vue';
import Uku from 'Uku';

const app = createApp();
app.use(Uku);
```
